QuickSort Algorithm

An elegant divide-and-conquer algorithm that selects a pivot element and partitions the array around it, recursively sorting the sub-arrays.

Time (Avg) O(n log n)
Time (Worst) O(n^2)
Space O(log n)

Live Execution

quickSort(arr, low, high) {
  if (low < high) {
    pi = partition(arr, low, high);
    quickSort(arr, low, pi - 1);
    quickSort(arr, pi + 1, high);
  }
}
